Что бы начать создать HTML тег по конструктору, нужно объявить пременую, которая будет ссылаться на конструктор. Выглядит это так.

let p = new CreateTag({ tag: "p", tagClass: "item", tagContent: "Hello, how are you?" });

В переменую нужно передать объект и в объекте дожны быть значения.

tag: "p" Название тега.

tagClass: "itmem" Класс тега.

tagContent: "Hello, how are you?" Содержимое тега.

Для того что бы вставить созданный тег в html, нужно вызвать один из нескольких методов.

p.begin();

Создает тег, но не вставляет в html, что бы вставить нужно также использовать метод

pastElementInDocument();

Этот медод вставляет тег в тег body.

Но если нужно втсавить в конкретный блок, то нужно указать его селектор в при создании объекта по конструктору и спользоавать значение parentPlace: "#wrapper" в кавычках указать селектор родительского тега.

И вставлять его нужно используя метод pastElemSelected();

Для того чтобы создать тег в и срзу в тавить его в тег html, нужно использовать методы beginAndPastDoc() или beginAndPastPlace().

beginAndPastDoc() вставляет тег только в html

beginAndPastPlace() вставляет тег в другой тег.

*Опять таки если нужно вставить тег в контреный тег нужно указать селектор родителя в при создании объекта.

Удаление тегов

Если нужно удалить тег из html, нужно использовать метод removeElemInDoc().

Но если у тега есть родитель нужно применить метод removeElemInPrarent().

Добавление событий

Для того что бы добаить событие на тег, нужно использовать метод getEvent().

Выглядит это вот так. btn.getEvent("click", functionName);

Этот метод принемает два аргумента, превый название события, а второй, имя фунции.

Удаление событий.

Для того чтобы удалить собыьтие с тега нужно использовать метод removeEvent();

Он принемает два аргумента, так же как и в добавлении, первый для нзвание события, второй имя функции

btn.removeEvent("click", functionName);
